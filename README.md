# Remove Duplicates from a String in Java

This Java program removes duplicate characters from a string while preserving the original order of appearance using LinkedHashSet.

## Features

- Removes duplicate characters
- Preserves insertion order
- Uses LinkedHashSet
- Beginner-friendly implementation
